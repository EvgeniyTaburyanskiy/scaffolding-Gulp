# gulp-scaffolding
Scaffolding for Frontend Build

### Базовый набор для разработки.

Технологический стек:
* SCSS
* JS (ES6 + Babel)
* PUG

### Scaffolding
.
   |-lib
   |---gulp
   |-----tasks
   |-src
   |---fonts
   |---img
   |-----favicon
   |-----icons
   |---js
   |-----modules
   |---style
   |-----base
   |-----helpers
   |-----modules
   |-----pages
   |-----vendors
   |---template
   |-----helpers
   |-----modules
   |-----pages



### Getting started:
```sh
mkdir Proj
cd ./Proj
git clone https://github.com/EvgeniyTaburyanskiy/scaffolding-Gulp.git .

npm install
gulp
```
