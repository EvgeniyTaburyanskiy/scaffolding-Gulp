# gulp-scaffolding
Scaffolding for Frontend Build

### Getting started:
```sh
mkdir Proj
cd ./Proj
git clone https://github.com/EvgeniyTaburyanskiy/scaffolding-Gulp.git .

npm install
gulp
```

### Tech Stack

* SCSS
* JS (ES6 + Babel + Browserify)
* PUG
